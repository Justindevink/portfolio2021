OBA Book Report

OBA Book Report is a concept for a webapp that helps children to make their book reports for school. OBA stands for Openbare Bibliotheek Amsterdam, which means Amsterdam Public Library in Dutch. This project focused a lot on microinteractions, which I added in the prototype through component states in Adobe XD. OBA Book Report made use of an API from OBA where metadata of every book available is stored, this made the function available that when the user made a new book report file, they could search for the book they used as a subject for the book report. Then OBA Book Report automaticaly filled in some data in the document, like the title and the author of the book. Lastly the user could follow a template that existed of chapters like preface, about the author, the story and conclusion, so that the user was guided while making the book report.

  • Year: 2020
  • Client: Amsterdam Public Library & HvA
